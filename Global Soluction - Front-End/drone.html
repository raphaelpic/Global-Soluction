Nosso projeto tem como foco principal de desenvolver uma tecnologia para monitoramento dos oceanos na coleta de dados para análise, seja para segurança, pesca, sendo assim operamos com Drones para captar tudo que ocorre na superfície dos oceanos, e infravermelho para identificar tudo que ocorre dentro e fundo do mar.

Nosso projeto visa auxiliar empresas que necessitam desse estudo e ampliar o conhecimento tecnológico.
